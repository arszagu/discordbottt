ЗАВИСИМОСТИ ПРОЕКТА
===================

Все зависимости указаны в файле requirements.txt

Основные библиотеки:
--------------------

1. discord.py (2.3.2)
   - Официальная Python библиотека для работы с Discord API
   - Позволяет создавать ботов, взаимодействовать с серверами
   - Документация: https://discordpy.readthedocs.io/

2. python-dotenv (1.0.0)
   - Загружает переменные окружения из файла .env
   - Используется для хранения конфиденциальных данных (токены)

3. matplotlib (3.8.2)
   - Библиотека для создания графиков и диаграмм
   - Создаёт красивые визуализации статистики
   - Поддерживает множество типов графиков (pie, bar, line и т.д.)

4. pillow (10.1.0)
   - Библиотека для обработки изображений
   - Требуется для корректной работы matplotlib
   - Используется для форматирования и обработки графиков

5. numpy (1.24.3)
   - Библиотека для численных вычислений
   - Поддерживает работу с массивами и матрицами
   - Требуется для mathplotlib и обработки данных

УСТАНОВКА ЗАВИСИМОСТЕЙ
=======================

Команда:
  pip install -r requirements.txt

Эта команда установит все необходимые версии пакетов.

ПРОВЕРКА УСТАНОВКИ
===================

Проверить, установлены ли зависимости:
  pip list

Установить конкретный пакет:
  pip install discord.py==2.3.2

Обновить пакет:
  pip install --upgrade discord.py

ВЕРСИИ И СОВМЕСТИМОСТЬ
=======================

Тестировалось на:
- Python 3.8+
- Windows 10+
- Linux (Ubuntu 20.04+)
- macOS

Если возникают проблемы совместимости, попробуй обновить pip:
  python -m pip install --upgrade pip

ПРОБЛЕМЫ И РЕШЕНИЯ
===================

Ошибка: "No module named 'discord'"
  → Активируй виртуальное окружение и установи зависимости

Ошибка: "ImportError: cannot import name 'Intents'"
  → Обнови discord.py: pip install --upgrade discord.py

Ошибка: "matplotlib backend issue"
  → Установлен backend Agg в config.py для Windows

Дополнительная информация в README.md
