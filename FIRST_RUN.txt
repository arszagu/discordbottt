⚡ БЫСТРЫЙ СТАРТ (САМОЕ ВАЖНОЕ!)

ШАГ 1: ПОЛУЧИ ТОКЕН DISCORD
────────────────────────────

1. Перейди на: https://discord.com/developers/applications
2. Нажми кнопку "New Application"
3. Придумай имя приложению (например "StatBot")
4. Нажми "Create"
5. Перейди на вкладку "Bot" (слева)
6. Нажми "Add Bot"
7. Нажми "Copy" под токеном (это и есть токен!)
8. Включи интенты:
   - Включи "MESSAGE CONTENT INTENT"
   - Включи "SERVER MEMBERS INTENT"
9. Сохрани изменения

ИМЕЕШЬ ТОКЕН? → ИДЁМ ДАЛЬШЕ ⬇️

═══════════════════════════════════════════════════════════════

ШАГ 2: ДОБАВЬ ТОКЕН В ФАЙЛ .env
────────────────────────────────

1. ОТКРОЙ файл .env (в этой папке discordbottt)
2. ТАМ ТЫ ВИДИШЬ:
   DISCORD_TOKEN=токен_бота

3. ЗАМЕНИ значение токена на твой реальный токен
   Получится примерно так:
   DISCORD_TOKEN=MzU1OTk5NzI5MzEzMDcxMDcyMw.DLgDEw.123456789abcdef

4. СОХРАНИ ФАЙЛ (Ctrl+S)

⚠️  ВАЖНО: Никогда не делись этим токеном!

═══════════════════════════════════════════════════════════════

ШАГ 3: ДОБАВЬ БОТА НА СВОЙ СЕРВЕР
──────────────────────────────────

1. Вернись в Discord Developer Portal
2. Перейди на "OAuth2" (слева)
3. Нажми "URL Generator"
4. Выбери скоп: bot
5. Выбери разрешения:
   ✓ Send Messages
   ✓ Embed Links
   ✓ Attach Files
   ✓ Read Messages/View Channels
6. Скопируй сгенерированный URL
7. Открой URL в браузере
8. Выбери СВОЙ сервер (не Discord сервер, а СВОЙ!)
9. Нажми "Authorize"
10. Пройди проверку безопасности (если нужно)
11. Готово! Бот добавлен на сервер! 🎉

❓ ЧТО ДЕЛАТЬ ЕСЛИ...

❌ "Invalid token"
→ Проверь, правильно ли ты скопировал токен в .env
→ Убедись, что нет пробелов в начале или конце
→ Попробуй скопировать токен ещё раз с Discord Developer

❌ "Bot not found"
→ Проверь, что ты добавил бота на сервер (ШАГ 3)
→ Убедись, что бот видно в списке участников сервера
→ Может быть нужно перезагрузить Discord

❌ "Bot не отвечает на команды"
→ Попробуй команду /help
→ Убедись, что в PowerShell было сообщение о успешном подключении
→ Перезагрузи бота и Discord

❌ "ModuleNotFoundError: No module named 'discord'"
→ Активируй виртуальное окружение:
   venv\Scripts\activate
→ Установи зависимости:
   pip install -r requirements.txt

❌ Другая ошибка?
→ Запусти проверку конфигурации:
   venv\Scripts\activate
   python check_config.py
→ Прочитай README.md в разделе "Решение проблем"
