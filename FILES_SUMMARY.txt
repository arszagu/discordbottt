🚀 ПОШАГОВАЯ ИНСТРУКЦИЯ
═══════════════════════════════════════════════════════════════════

ШАГ 1 - ПОЛУЧИ ТОКЕН
  1. Перейди на https://discord.com/developers/applications
  2. Нажми "New Application" и придумай имя
  3. Перейди на "Bot" → "Add Bot"
  4. Скопируй токен
  5. Включи интенты: Message Content Intent, Server Members Intent

ШАГ 2 - ДОБАВЬ ТОКЕН В .env
  1. Открой файл .env (в этой папке)
  2. Замени значение токена на скопированный реальный токен Discord
  3. Сохрани файл

ШАГ 3 - УСТАНОВИ ЗАВИСИМОСТИ
  1. Открой PowerShell в этой папке
  2. Выполни: venv\Scripts\activate
  3. Выполни: pip install -r requirements.txt

ШАГ 4 - ЗАПУСТИ БОТА
  Способ 1 (простейший):
    → Дважды нажми run.bat
  
  Способ 2 (через PowerShell):
    → python main.py

ШАГ 5 - ДОБАВЬ БОТА НА СЕРВЕР
  1. В Discord Developer Portal переходишь в OAuth2 → URL Generator
  2. Выбираешь скоп "bot"
  3. Выбираешь разрешения (Send Messages, Embed Links, Attach Files)
  4. Копируешь сгенерированный URL
  5. Переходишь по URL и добавляешь бота на сервер

ШАГ 6 - ТЕСТИРУЙ КОМАНДЫ
  На любом канале сервера пиши:
    /help           → Справка
    /stats          → Статистика
    /member_graph   → Графики

Q: Где взять токен бота?
A: https://discord.com/developers/applications

Q: Как добавить бота на свой сервер?
A: https://discord.com/developers → OAuth2 → URL Generator

Q: Когда я пишу команду, бот не отвечает
A: 1) Проверь, что токен правильно вставлен в .env
   2) Проверь, что бот добавлен на сервер
   3) Проверь, что у бота есть права

Q: Ошибка "No module named 'discord'"
A: Активируй venv: venv\Scripts\activate
   Установи зависимости: pip install -r requirements.txt

Q: Как я могу добавить свои команды?
A: 1) Создай новый файл в папке cogs/ (например commands.py)
   2) Скопируй структуру из stats.py
   3) Добавь свой класс и команды
   4) Перезапусти бота

Q: Где я могу изменить цвета графиков?
A: В файле config.py в секции COLORS
