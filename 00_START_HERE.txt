🚀 НАЧАЛО РАБОТЫ (3 ШАГА)
══════════════════════════

ШАГ 1️⃣  - ОТРЕДАКТИРУЙ ТОКЕН
───────────────────────────────
Открой файл .env и замени:
  DISCORD_TOKEN=токен_бота
на свой реальный токен Discord бота

Где взять токен?
  → https://discord.com/developers/applications
  → New Application → Bot → Copy Token

ШАГ 2️⃣  - УСТАНОВИ ЗАВИСИМОСТИ
──────────────────────────────────
В терминале (PowerShell) выполни:
  
  venv\Scripts\activate
  pip install -r requirements.txt

ШАГ 3️⃣  - ЗАПУСТИ БОТА
────────────────────────
Выбери один из способов:

  Способ 1 - Двойной клик (самый простой):
    👉 Дважды нажми на файл run.bat

  Способ 2 - Через терминал:
    venv\Scripts\activate
    python main.py

❓ ЧАСТЫЕ ВОПРОСЫ
═════════════════

Q: "Где найти токен бота?"
A: https://discord.com/developers/applications

Q: "Как добавить бота на сервер?"
A: https://discord.com/developers/applications → OAuth2 → URL Generator

Q: "Бот не отвечает"
A: Проверь, что он добавлен на сервер и имеет права

Q: "Где включить Intents?"
A: discord.com/developers → Bot → Privileged Gateway Intents

Q: "Можно ли добавить свои команды?"
A: Да! Создай новый файл в папке cogs/ и добавь класс с командами
